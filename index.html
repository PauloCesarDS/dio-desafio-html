Base

O HTML (aquela coisa que você usa para criar páginas da web) é o que dá a estrutura básica para tudo na internet. Pense nele como o esqueleto da página. Com o HTML, você pode dizer ao navegador o que é um título, um parágrafo, um link ou uma imagem. É como usar etiquetas para organizar as coisas e mostrar ao seu navegador como elas devem ser exibidas.

Princípios

Alguns princípios legais do HTML incluem fazer as coisas de maneira que façam sentido (é o que chamamos de "semântica"). Também é importante tornar as páginas acessíveis para todos, incluindo pessoas com deficiências. Além disso, é preciso ter certeza de que as coisas funcionem direito em diferentes navegadores, para que ninguém fique de fora.

Evolução

O HTML está sempre mudando e melhorando. O HTML5 é a versão mais moderna, que trouxe muitas coisas bacanas, como suporte para vídeos e áudios, e elementos mais espertinhos que explicam o que é o quê. Além disso, o HTML5 deixou o desenvolvimento web mais fácil e permitiu criar aplicativos web super maneiros. Então, fique atento às novidades para criar sites e apps cada vez mais legais.
